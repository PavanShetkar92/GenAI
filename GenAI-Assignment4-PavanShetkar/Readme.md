# GenAI-Assingment_4-PavanShetkar

## Project Overview

This assignment demonstrates the use of Python File Handling operations including:

- Creating files
- Writing data to files
- Reading data from files
- Appending data to existing files
- Using file modes (`r`, `w`, `a`)
- Exception handling during file operations
- Processing numerical and text data from files
- Generating reports using file handling

The implementation follows the assignment restrictions and uses only basic Python concepts without advanced libraries.
