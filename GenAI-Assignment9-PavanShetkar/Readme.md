# GenAI-Assingment_9-PavanShetkar

**#Problem Statement**
Perform basic mathematical and statistical operations using **NumPy** (and optionally SciPy).  
This assignment focuses on:
- NumPy mathematical operations
- NumPy statistical operations
- Basic data analysis (mean, median, variance, standard deviation, etc.)

---

# Tasks

### Task 1: Creating NumPy Arrays
- Create arrays from lists, ranges, random numbers, and 2D structures.
- Print all arrays.

### Task 2: Mathematical Operations
- Addition, subtraction, multiplication, division, power.

### Task 3: NumPy Mathematical Functions
- Square root, exponential, logarithm
- Sum and average of elements

### Task 4: Aggregation Operations
- Minimum, maximum, mean, median, standard deviation, variance

### Task 5: Statistical Operations
- Mean, median, mode, variance, standard deviation, range, percentile
- Graph visualization

### Task 6: Percentile & Sorting
- Calculate 25th, 50th, 75th percentiles
- Sorting
- Count students scoring above average

### Task 7: Mini Use Case – Sales Analysis
- Mean, median, standard deviation, variance
- Identify sales above average
- Graph visualization
